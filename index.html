HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • P is used to create a new paragraph.
  • Br is used to create a break in the page.
  • Each statement needs an opening and a closing.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can change the color with the identifier code.
  • You need to link the style css page to the head of your code in order for it to be recognized.
  • Think of CSS as a box diagram and padding is the space surrounding that box and margin is the space your box is from the edge of the page.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript reads code top to bottom in a linear fashion. Code needs to be in order.
  • You can used conditional statements and loops to control the order in which the code is executed.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision-making into our code.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal.
